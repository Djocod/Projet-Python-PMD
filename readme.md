# JMJ Festival Game

## Overview
J.M.J Festival is a classic text-based adventure game built entirely in standard Python. 
It challenges you to navigate a funny, festival landscape by making choices where to go and interacting with the people you will encounter on your way What sounds easy can quickly become tricky. 
Be aware that not everyone who you will meet on your way has your best interest in mind. 
Do not miss opportunities to collect items that will help you to either to stay strong or will be useful for your encounters. 
The goal is to make it to the main stage. 
The game is designed to run directly within your terminal or command prompt, offering a retro, command-line experience.

## Features
- **Player Inventory**: Manage objects, potions, and cashless items to survive and progress.
- **Combat System**: Fight against Drunk Crowds and Security using unique attacks and items.
- **Exploration**: Navigate through different rooms on the map and collect objects.
- **Dynamic Events**: Encounter random events and challenges as you move through the festival.
- **Interactive Story**: Make choices that affect your journey and interactions with characters.

## How to Play
1. **Start the Game**: Run the `main.py` file to begin.
2. **Navigate the Map**: Use the `move()` function to explore the festival.
3. **Manage Inventory**: Collect and use objects and potions strategically.
4. **Engage in Battles**: Use attacks and items to defeat enemies.
5. **Reach the Main Stage**: Overcome all challenges to enjoy the festival's music.


   ```bash

   ```
3. Run the game:

   ```bash
   python main.py
   ```



## File Structure
- `main.py`: Entry point of the game.
- `function.py`: Contains core game functions such as movement, combat, and map plotting.
- `data_game.py`: Stores game data including player stats, enemy stats, and objects.
- `readme.txt`: Project documentation.
